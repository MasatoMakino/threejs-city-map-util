Class JapanStandardRegionalMeshUtil

総務省統計局制定の基準地域メッシュコードを、緯度経度と相互変換するためのクラスです。

See

  • https://www.stat.go.jp/data/mesh/m_tuite.html 基準地域メッシュコードは、世界測地系での緯度経度を基準に算出されます。
  • https://www.stat.go.jp/data/mesh/pdf/gaiyo1.pdf 基準地域メッシュコードの概要図はこちらを参照ください。
  • https://www.stat.go.jp/data/mesh/pdf/m-kukaku.pdf

PLATEAUプラットフォームはファイル命名形式に基準地域メッシュコードを利用しています。

  • https://www.mlit.go.jp/plateau/learning/tpc03-1/#p3_1 インタラクティブマップで目的地のメッシュコードが確認できます。
  • https://jstatmap.e-stat.go.jp/jstatmap/main/trialstart.html
